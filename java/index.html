      • Java从入门到实战

      Post on 2015-01-01 11:11

      本文的目的是把对于没有编程基础的朋友，快速带入编程的世界，如果你是已经会编程的朋友，可以直接忽视本文。

      全文会引入一系列文章，包括Java基本的开发环境搭建到开发出一套可增删改查的Java Web项目，及一个简单的爬虫。这样你可以了解到关于Java Application 和 Java Web Application大致的相关知识。

      我相信当你按文章一步一步完成的时候，所得到的那份成就感一定会不停的激励你前进。

      1. 在Ubuntu下安装Java开发环境
      2. Java项目管理工具Maven
      3. 快速创建一个Java Web项目
      4. 如果编写一个简单的Web爬虫 - demo
      5. 笑泥马 - demo
      6. Q宠大乐斗辅助 - demo

      以上所有项目都用到了我自己定的类库与框架，大大减少代码的编写量，效率提升那是刚刚的。需要下载代码的请移步到Github，已经发布的Project可以到Maven中央库查看。
