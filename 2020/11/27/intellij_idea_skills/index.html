      Intellij IDEA 使用技巧

      Post on 2020-11-27 16:27

      适用于版本 IntelliJ IDEA 2019.1.4 (Ultimate Edition), 其它版本未测试。
      操作系统 macOS

      开启命名行启动器

      打开IDEA, 点击 Tools->Create Command-Line Launcher... 激活命令行启动功能。

      • idea . 以项目形式打开当前目录
      • idea pom.xml 打开maven项目
      • idea diff <left> <right> 运行比较工具，用于比较左右两个文本的不同
      • idea merge <local> <remote> [base] <merged> 合并

      配置

      Java Import

      通过Preference -> Edit -> General -> Auto import进行配置, 勾上

      • Add unambiguous imports on the fly
      • Optimize imports on the fly (for current project)
